## Features

- **Per-Site Time Limits**  
  Set custom daily limits for any website. Decide exactly how long you want to spend per site.  

- **Real-Time Tracking**  
  Live counters update every second to show exactly how much time you've spent on each site.  

- **Smart Blocking**  
  When time’s up, the site is blocked instantly with a motivational page. No more "just one more minute."  

- **Early Warnings**  
  Get notified at 80% of your limit. A floating badge shows remaining time on tracked pages.  

- **Strict Mode**  
  Make limits unbreakable — no overrides, no escape. Perfect for serious focus sessions.  

- **Daily Auto-Reset**  
  Limits reset automatically at midnight. Track weekly trends with 7-day history.

## How It Works

1. **Install Extension**  
   Add Focus Timer to Chrome in one click. No account needed. Everything stays on your device.  

2. **Set Your Limits**  
   Click the extension icon, hit “Add Limit for This Site,” and set your daily limit in seconds. Choose a color tag.  

3. **Focus & Succeed**  
   Browse normally. Focus Timer tracks time automatically and blocks sites when limits are reached. Check daily stats to improve.

## Installation

1. Download the extension ZIP file.  
2. Open `chrome://extensions/` in Chrome.  
3. Enable Developer mode.  
4. Unizp the file and Drag and drop the folder.  
5. Start focusing!
